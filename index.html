Introduction & Architecture

Bienvenue dans ce guide complet. Nous allons construire ensemble une infrastructure moderne, sécurisée et automatisée sur un VPS Ubuntu.

Concept Global

L'objectif est d'utiliser **Docker** pour isoler nos services, **Dockge** pour les gérer visuellement, et **Cloudflare Tunnel** pour les exposer de manière sécurisée sans ouvrir de ports sur le pare-feu du VPS.

Schéma Réseau

[ Internaute ] <---> [ Cloudflare Edge ]
                           |
                   ( Tunnel Chiffré )
                           |
                  [ Connecteur Docker ] --- (Réseau Interne)
                                                  |
                                          [ WordPress Service ]
                                                  |
                                          [ Dockge Manager ]

Sécurité

Aucun port ouvert (Zéro-Trust). HTTPS géré nativement.

Flexibilité

Déploiement en un clic avec Docker Compose & Dockge.
